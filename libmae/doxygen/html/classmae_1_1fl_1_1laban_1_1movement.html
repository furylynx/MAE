Creates a movement element which represents a movement symbol in the score (e.g. direction and level or a turn) and (if present) a hold sign.

Parameters
columnThe column to which this symbol is meant to be added.
measureThe measure in which this symbol starts (and therefore the motion).
beatThe beat (decimals allowed) of the measure which is the beginning of the motion.
durationThe duration of the symbol (given in an amount of beats).
symbolThe symbol defining the kind of movement (e.g. direction+level or turn)
holdIf true a hold sign is attached to symbol.
pre_signA pre-sign which can be used to address another body part than the one defined by the column.
