mae::fl::fl_movement_controller::fl_movement_controller ( unsigned int  pose_buffer_size = 0,
double  framerate = 1.0/30.0,
bool  debug = false 
)

Creates a new controller for movements analysis based on Labanotation and fl_skeletons. The used body parts are assumed to be the default ones used by the NITE skeleton. Since no column definition is given the default columns are the only ones that can be used.

Parameters
pose_buffer_sizeThe buffer size for the movement detector (number of frames to process). Values <= 1 means calculating the buffer size from the registered sequences.
framerateThe framerate for the stream.
debugTrue for debug console print.
