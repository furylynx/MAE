mae::fl::laban::ps::digit_part::digit_part ( e_digit  digit,
unsigned int  knuckle 
)

Creates a digit pre-sign which consist of a digit (finger or toes) and a knuckle.

Parameters
digitThe addressed digit.
knuckleThe addressed knuckle. Zero represents the whole digit, whereas A number represents the knuckle counted from the base. Therefore 3 for toes and 4 for fingers is the maximum valid value.
