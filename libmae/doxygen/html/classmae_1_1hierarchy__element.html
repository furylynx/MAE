mae::hierarchy_element::hierarchy_element ( int  id,
std::string  name,
bool  torso_joint = false,
bool  dummy = false 
)

Creates a new hierarchy element. Elements are defined by their id and have a name that is printed whenever needed (e.g. for bvh export).

Please note that at least three torso joints must be defined in order to use the skeleton for the FL* motion controller.

Parameters
idThe id of the element.
nameThe name of the element.
torso_jointTrue if the joint is a torso joint.
dummyTrue if the joint is a dummy joint.
