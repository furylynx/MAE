std::shared_ptr< i_movement >  dist_to,
std::shared_ptr< i_movement >  check 
)
protectedvirtual

Checks the distance between the first to parameters and then examines whether the third parameter is matching. This is used to check whether the start pose does also match. Therefore distance between the two elements of the streamed sequence (potential first movement dist_from and potential start pose dist_to) are compared to the position of the first movement in the registered sequence (set value).

Parameters
dist_fromThe distance from.
dist_toThe distance to.
checkThe element to be checked to have enough distance.
Returns
True if distance is okay.
